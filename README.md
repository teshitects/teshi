# Teshi
`Teshi` is a program built for testers

## Roadmap

- [ ] **Test Case Management** (import, search, add, update, modify, delete, export, and sync test cases)
- [ ] **API Test Generator and Runner** (use `playwright` to record manual steps and network requests, generate automated scripts, and run them)
- [ ] **Other Useful Tools** (e.g., JSON diff tool, scheduled tasks, AI suggestions)
